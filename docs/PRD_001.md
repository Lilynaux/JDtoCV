
# PRD 001

## Project Overview

ResumeTailor 是一个基于 OpenAI API 的本地 Python 工具。

目标：

根据 Job Description (JD) 自动分析岗位需求，并基于个人知识库生成定制化中英文简历。

项目以本地文件为核心，不开发 Web 系统。

所有简历最终输出为 LaTeX 文件，方便在 VS Code 中继续编辑并导出 PDF。

⸻

## Background

目前求职过程中需要：

* 阅读 JD
* 提取关键词
* 修改简历
* 翻译中英文
* ATS 优化

每次投递都需要重复操作。

目标是建立一个可复用的简历知识库。

⸻

## Product Goal

输入：

JD

输出：

1. JD 分析报告
2. 匹配度分析
3. 中文定制简历
4. 英文定制简历
5. LaTeX 简历文件
6. PDF 简历

整个流程控制在 1 分钟内完成。

另外我建议你让 CC 顺手帮你解决 LaTeX 中文环境问题。

你现在大概率是：

pdflatex

所以中文会报错。

直接换成：

xelatex

然后模板头部改成：

\documentclass[UTF8]{ctexart}

或者如果继续用 Jake’s Resume：

\usepackage{fontspec}
\setmainfont{Times New Roman}
\setCJKmainfont{Noto Serif CJK SC}

这样以后：

* resume_cn.tex
* resume_en.tex

都能在 VS Code 一键导出 PDF。

这比 Word + WPS 稳定得多，也更适合版本管理和 GitHub。
