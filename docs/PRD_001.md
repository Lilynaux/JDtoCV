# PRD_001: ResumeTailor V1（核心产品需求）

## **1. Project Overview（项目概述）**

ResumeTailor 是一个基于 OpenAI API 的本地 Python 工具，目标是根据职位描述（JD）自动分析岗位需求，并基于个人知识库生成定制化中英文简历。

项目以本地文件为核心，不开发 Web 系统；所有简历最终输出为 LaTeX 文件，方便在 VS Code 中继续编辑并导出 PDF。

---

## **2. Background（背景）**

求职过程中，每次投递都需要重复以下操作：

- 阅读 JD
- 提取关键词
- 修改简历
- 翻译中英文
- ATS 优化

目标是建立一个可复用的简历知识库，把这些重复劳动自动化。

---

## **3. Product Goal（产品目标）**

**输入**：JD

**输出**：

1. JD 分析报告
2. 匹配度分析
3. 中文定制简历
4. 英文定制简历
5. LaTeX 简历文件
6. PDF 简历

整个流程控制在 1 分钟内完成。

---

## **4. User Workflow（用户工作流）**

1. 准备 JD（支持 `.txt` / `.pdf` / `.docx`）
2. 运行脚本 `python main.py`
3. AI 分析 JD，输出岗位类别、核心技能、ATS 关键词、缺失能力
4. 匹配知识库
5. 生成定制简历
6. 导出 LaTeX
7. 在 VS Code 手动修改
8. 编译 PDF

---

## **5. Core Features（核心功能）**

### Feature 1 — JD Analyzer

**输入**：JD

**输出**：

- Position Summary
- Required Skills
- Preferred Skills
- ATS Keywords
- Missing Skills
- Match Score（0–100）

**输出文件**：`outputs/jd_analysis.md`

### Feature 2 — Knowledge Base

建立长期知识库，目录结构：

```text
knowledge_base/
├── experience/      # 实习经历，例如 EY.md / Citic.md / Audit.md
├── projects/        # 项目经历，例如 TradingSystem.md / OpenAlex.md / Dissertation.md
├── skills/          # 技能库，例如 Python.md / CreditRisk.md / ProductManagement.md
└── terminology/      # 行业术语库，例如 FinTech.md / RiskManagement.md / ProductManager.md
```

### Feature 3 — Resume Optimizer

**输入**：Master Resume、Knowledge Base、JD

**输出**：针对目标岗位优化后的简历

**原则**：

- 不得编造经历
- 只能重组和优化表达

### Feature 4 — CN → EN Synchronization

中文简历为主版本，用户主要编辑中文。流程：Chinese Resume → AI Translation → English Resume，同步更新。

### Feature 5 — English Override

允许手动优化英文表达，该修改仅保留在英文版本，不反向同步回中文。例如：

| 阶段 | 内容 |
| --- | --- |
| 中文原文 | 参与AI Agent开发 |
| 自动翻译 | Participated in AI Agent development |
| 用户修改 | Designed and implemented AI-powered financial agents |

### Feature 6 — Resume Diff

输出新增、删除、修改的内容，方便人工审核。

## **7. LaTeX Requirements（LaTeX 排版要求）**

最终简历使用 LaTeX，原因：

- 格式统一
- ATS 友好
- 便于版本控制
- VS Code 编辑体验好

推荐模板：Jake's Resume 或 AwesomeCV。

### Chinese LaTeX Support

当前环境疑似仅支持英文排版，需要解决 XeLaTeX 中文编译配置，目标是同时支持中文简历、英文简历、中英混排。

**建议方案**：

- 编译器使用 `xelatex`（而非 `pdflatex`，后者无法正确渲染中文）
- 中文字体推荐 Noto Serif CJK SC / Noto Sans CJK SC
- 模板头部使用：

  ```latex
  \documentclass[UTF8]{ctexart}
  ```

  或在 Jake's Resume 模板基础上添加：

  ```latex
  \usepackage{fontspec}
  \setmainfont{Times New Roman}
  \setCJKmainfont{Noto Serif CJK SC}
  ```

**编译命令**：

```bash
xelatex resume_cn.tex
```

实现后，`resume_cn.tex` 与 `resume_en.tex` 均可在 VS Code 中一键导出 PDF，相比 Word + WPS 更稳定，也更适合版本管理和 GitHub 协作。
